# url-shortener

URL Shortener – Node.js + Express + PostgreSQL + EJS (MVC Architecture)

A lightweight, fast, and easy-to-use URL Shortening Web Application similar to Bit.ly.
Users can create short URLs, view analytics, copy links, and delete them — all through a clean dashboard interface.

Features
Short URL Generation

Create short links for long URLs

Optional custom short code (6–8 characters)

Statistics Dashboard

View link details

Track total clicks

See last clicked time

View creation timestamp

User-Friendly UI

TailwindCSS for styling

EJS for server-side rendered pages

Clean Architecture (MVC)

Models → Sequelize + PostgreSQL

Views → EJS templates

Controllers → Express Routes

Static Assets → /public/js/\*

Tech Stack
Layer → Technology
Backend → Node.js, Express
Frontend → EJS, Tailwind CSS
Database → PostgreSQL (via Sequelize ORM)
Architecture →MVC
Deployment → Render and Neon
